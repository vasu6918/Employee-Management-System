# Employee Management System

A simple Python-based Employee Management System using functions and JSON file handling.

## Features

- Add Employee
- View Employees
- Search Employee
- Update Employee
- Remove Employee
- JSON data storage
- Input validation using try-except

## Technologies Used

- Python
- JSON

## Concepts Used

- Functions
- Lists
- Dictionaries
- Loops
- Conditional Statements
- Exception Handling
- File Handling

## How to Run

```bash
python employee_management.py
```
